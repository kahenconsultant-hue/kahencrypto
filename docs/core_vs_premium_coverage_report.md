# Core vs Premium Coverage Report

## Core Coverage

Core coverage شامل مواردی است که برای تحلیل کاربردی پایه کافی‌اند:

- قیمت و حجم BTC/ETH/SOL
- DXY/US10Y/Nasdaq/Gold proxy
- stablecoin supply proxy
- RSS خبر و policy events
- freshness و source health

## Premium Coverage

Premium coverage فقط برای تحلیل‌های پیشرفته استفاده می‌شود:

- ETF direct flow
- exchange reserves
- deep on-chain
- derivatives/open interest حرفه‌ای
- whale attribution
- institutional feeds

## قانون تصمیم

کمبود premium نباید score کل سیستم را نابود کند. فقط confidence همان ماژول یا premium notices را تغییر می‌دهد.
