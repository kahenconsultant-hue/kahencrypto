# Proxy Alert Rules

## Macro Pressure Proxy Alert

اگر DXY و US10Y فشارزا باشند و BTC/ETH/SOL یا Nasdaq ضعف نشان دهند، هشدار فشار کلان proxy تولید می‌شود.

## Liquidity Proxy Alert

اگر stablecoin supply، spot volume و market trend هم‌راستا شوند، liquidity proxy alert فعال می‌شود. اگر حرکت قیمت فقط با futures/volume ناپایدار باشد، هشدار با confidence پایین‌تر نمایش داده می‌شود.

## Stablecoin Pressure Alert

اگر stablecoin liquidity signal منفی شود و هم‌زمان volume یا price trend ضعف نشان دهد، هشدار فشار استیبل‌کوین تولید می‌شود.

## Data Degradation / Premium Notice

اگر core reliability افت کند، data degradation alert تولید می‌شود. اگر فقط premium coverage پایین باشد، premium missing notice تولید می‌شود.
