# Regime Proxy Rules

## قواعد اصلی

- Macro pressure وقتی فعال می‌شود که macro pressure proxy زیر آستانه منفی باشد.
- Liquidity contraction proxy وقتی فعال می‌شود که crypto liquidity یا stablecoin liquidity منفی شود.
- Volatility expansion وقتی فعال می‌شود که volatility proxy فشار بالا نشان دهد.
- Risk-on فقط وقتی پذیرفته می‌شود که macro و liquidity هم‌زمان حمایتی باشند.
- Neutral/mixed وقتی انتخاب می‌شود که سیگنال‌ها هم‌راستا نیستند.

## توضیح فارسی

هر regime باید drivers و missing inputs را نشان دهد. اگر ETF یا derivatives premium موجود نیست، باید صریحاً گفته شود که regime بر اساس free-data-plus-proxies ساخته شده است.
