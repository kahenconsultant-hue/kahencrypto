# Remaining Data Gaps

## Premium gaps

- ETF direct flow
- CoinGlass derivatives/open interest کامل
- Glassnode/CryptoQuant exchange reserves
- Whale Alert attribution
- deep on-chain metrics
- Reuters/Bloomberg realtime feeds

## Core gaps that need monitoring

- RSS endpoint failures for individual sources
- Macro proxy provider availability
- stablecoin endpoint freshness
- websocket/runtime market data continuity

## Rule

این gapها نباید با عدد جعلی پر شوند. هر gap باید در source health، reliability و dashboard شفاف باشد.
