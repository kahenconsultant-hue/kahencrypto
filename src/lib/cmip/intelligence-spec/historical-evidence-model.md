# Historical Evidence Model

Historical evidence is contextual evidence. It is not prediction.

Required concepts:

- event definition
- sample selection
- sample size
- forward horizon
- median return
- positive outcome rate
- drawdown
- dispersion
- regime compatibility
- structural similarity
- structural difference
- sample limitation
- data availability bias

Rules:

1. Historical evidence must never be described as prediction.
2. Statistical claims require sample size.
3. Small samples must be labeled.
4. Historical comparison must separate similar conditions from different conditions.
5. Historical evidence may produce `confirmed`, `partially_confirmed`, `not_confirmed`, `contradicted`, or `insufficient_data`.
6. No historical claim may be generated from model memory alone.
7. Historical evidence must come from stored or calculated data.

Historical evidence may strengthen, weaken, or reject a hypothesis only after the current event definition is compared with the historical event definition.
